# User Manual

<p>The system aims to simplify academic and administrative management by providing the following key features:

Role-based authentication and access control.

Real-time task tracking for instructors.

Coordinator dashboard with DataTables and live charts.

Admin panel for managing users and assigning roles.

Responsive interface with a professional, green-gradient theme.</p>
